| Term                         | Definition                                                                                                |
| ---------------------------- | --------------------------------------------------------------------------------------------------------- |
| Authentication               | The process of verifying who a user is                                                                   |
| Authorization                | The process of determining what an authenticated user is allowed to do.                                  |
| Client                       | A person or organization using the services of another professional person or company.                    |
| Collision                    | A situation where two or more drones occupy the same physical space at the same time.                     |
| CRM                          | Abbreviation for Customer Relationship Management                                                        |
| CRM Collaborator             | This user assists the CRM Manager in client-related tasks.                                                |
| Customer Relationship Management | Practices, strategies, and technologies that companies use to manage and analyze customer interactions and data throughout the customer lifecycle itself. |
| Domain Specific Language     | A language for describing figures/shows.                                                                  |
| Drone Tech                   | This user is responsible for managing drones and the technical aspects of show preparation.               |
| Drones                       | All aircraft designed to fly without a pilot on board.                                                   |
| DSL                          | An abbreviation for Domain Specific Language                                                             |
| Elemental Volume             | Cubic meter used to verify collisions in simulations.                                                      |
| Exclusivity                  | A client's option to have a figure only available to them.                                                |
| Figure                       | Static or Dynamic formation of drones in the sky.                                                         |
| Show                         | A sequence of figures performed by a group of drones.                                                     |
| Show Designer                | This user is responsible for designing new figures to be used in drone shows.                              |
| Show Proposal                | A document presented to the client, detailing the planned show.                                           |
| Show Request                 | Client's initial request for a drone show, containing their specific requirements and preferences.         |
| Show Testing                 | Verifying a drone show's safety and quality through a simulation in a 3D space.                            |
| Shodrone                     | A company that provides customized drone multimedia shows.                                                |
| Simulation                   | Process of virtually testing a figure or a show before it is performed by the drones.                    |
